# Bug Hunter

**Ein Retro-Shooting-Gallery-Spiel, entstanden im Rahmen des GameJams der HHU des Jahres 2026!**

## Spielname
Bug Hunter

## Genre
Shooting Gallery Game

## Kurzbeschreibung des Spielziels und Spielinhalts
**Das Ziel:** So viele Punkte wie möglich sammeln, durch Präzision und Geschwindigkeit.

**Hintergrund zum Spiel:** Dieses Spiel wurde für einen GameJam mit dem Motto **"Retro"** entwickelt und ist stark von dem NES-Klassiker *Duck Hunt* inspiriert.

*Fun Fact aus der Entwicklung:* Ursprünglich gab es viel größere Pläne. Der  Arbeitstitel lautete "Duck Doom", es sollte das Arsenal an Waffen aus dem originalen *Doom* implementiert werden, kombiniert mit verschiedenen Leveln und Belohnungssystemen für Highscores. Die bittere Realität des Grafikdesigns (und die tickende Uhr des Game Jams) haben einen jedoch sehr schnell auf den Boden der Tatsachen zurückgeholt. Aus "Duck Doom" wurde so unser, aufs wesentliche reduzierter "Bug Hunter".

## Systemanforderung & Betriebssystem
* **Betriebssystem:** Plattformunabhängig (Windows, macOS, Linux, ChromeOS etc.)
* **Systemanforderungen:** Es gibt keine speziellen Hardwareanforderungen. Es wird nur ein moderner Webbrowser und eine Maus benötigt.
## Installation

**Keine Installation notwendig!** Bug Hunter läuft direkt in deinem Webbrowser. Einfach auf den folgenden Link klicken, um Bug Hunter zu spielen.
> **[Hier klicken, um Bug Hunter zu spielen](https://commanderexodus.github.io/GameJam2026/)**
